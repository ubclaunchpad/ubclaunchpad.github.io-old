Team Structure

Executive

Tech Lead

  • Experienced in some form of development
  • Researches the technical aspects of a project and organizes the tasks for the subteams
  • Responsible for helping team members with technical issues and hosting workshop sessions

Co-Captains

  • Plan the overall trajectory of the team and ensures successful the initiation of the bi-annual app project
  • Organize the weekly team meetings
  • Contacts external companies, students teams/clubs, and sponsors to organize potential events and sponsorships

Business Lead

  • Required to be entrepreneurially minded and familiar with the iOS, Android, and/or web app market
  • Develops the business strategy for the creation and distribution of the app along with the business sub team
  • Manages the team's financial planning and budget

Front-End

Front-End / UI Lead

  • Experienced in some form of front-end app development, be it web, iOS, or Android
  • Works closely with the designer and front-end team to write the front-end code
  • Creates the various sub-tasks that are required for front-end app development

Front-End Developers

  • Responsible for writing the front-end code along with the Front-End / UI Lead and other front-end developers
  • Examples include the client-side JavaScript in a web app or the views and controllers of a mobile app

Designer

  • Works with the front-end team to create an aesthetically pleasing design for the app
  • Will use Photoshop or equivalent software to design an intuitive user-friendly interface

Back-End

Back-End Lead

  • Required to be extremely experienced in back-end web frameworks or Parse or equivalent APIs on mobile
  • Along with the back-end team, will write the back-end code
  • Will create the various sub-tasks that are required for back-end app development

Back-End Developers

  • Responsible for writing the back-end code along with the Front-End / UI Lead and other front-end developers
  • Examples include Rails or Django code on web apps or API calls on mobile

Business

Business Development

  • Performs market analysis for potential apps
  • Regularly performs user testing and feedback
  • Liasons with the tech team to improve the app

Marketing

  • Responsible for the marketing strategy for both the student team and the app
  • Designs and creates the marketing materials for both the team and the project in development
